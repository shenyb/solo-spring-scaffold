# solo-spring-scaffold 项目状态 & 商业分析

## ✅ 已完成 P0 功能

### 1. Security 插件 (`plugins/security/`)
- SecurityConfig — 无状态 JWT 会话、CSRF 关闭、路由权限规则
- JwtTokenProvider — HMAC-SHA256 签名、access/refresh 双令牌
- JwtAuthenticationFilter — Bearer Token 提取 + SecurityContext 设置
- JwtAuthenticationEntryPoint — 401 JSON 响应（非重定向）
- SecurityExceptionHandler — 403 Forbidden JSON 响应
- TokenPair — 令牌对数据类
- AuthController — /api/auth/login, /register, /refresh
- AuthService/AuthServiceImpl — 登录/注册/刷新业务逻辑
- LoginRequest/LoginResponse/RegisterRequest DTO
- Flyway V3 迁移 — 添加 password 字段 + BCrypt 默认密码
- validation.properties 补丁 — 认证消息国际化
- application.yml 补丁 — JWT 配置项
- POM 依赖补丁 — spring-boot-starter-security + jjwt 0.12.6

### 2. Docker 插件 (`plugins/docker/`)
- Dockerfile — eclipse-temurin JRE Alpine、非 root 用户、分层构建、HEALTHCHECK
- docker-compose.yml — App + MySQL 服务、健康检查、网络隔离
- .dockerignore — 排除无关文件减小构建上下文

### 3. Test 插件 (`plugins/test/`)
- UserServiceTest — 单元测试（Mockito + Nested + DisplayName）
- UserControllerTest — 切片测试（MockMvc + @WebMvcTest）
- UserMapperTest — 集成测试（H2 内存库 + @DataJdbcTest）

### 4. Redis 插件 (`plugins/redis/`)
- RedisConfig — JSON 序列化 + Java 8 时间类型 + 30 分钟 TTL
- application-redis.yml — Lettuce 连接池配置
- POM 依赖补丁 — spring-boot-starter-data-redis + commons-pool2

### 5. Actuator 插件 (`plugins/actuator/`)
- DatabaseHealthIndicator — 自定义数据库健康检查
- application-actuator.yml — 暴露 health/info/metrics/prometheus 端点
- POM 依赖补丁 — spring-boot-starter-actuator + micrometer-registry-prometheus

## 🏗️ 插件体系架构

### plugin.yaml 规范
- name/version/description/author/category/tags
- requires — 插件间依赖
- templates — 新增文件列表
- patches — 补丁类型：yaml-append / properties-append / copy / dependencies / field-inject

### CLI 支持的插件命令
- `--plugins security,docker,test` — 启用插件
- `list-plugins` — 列出可用插件

### 补丁引擎
- YAML 追加：合并到 application.yml
- Properties 追加：合并到 validation.properties
- 依赖注入：解析 `module=g:a:v` 格式，注入到 pom.xml
- 代码注入：在指定行后插入 Java 字段/方法
